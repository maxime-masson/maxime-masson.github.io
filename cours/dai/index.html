DAI - Développement d'Applications Internet

Responsable d'UE: Maxime Masson

Organisation

il s’agit dans ce cours d’approfondir la connaissance du langage JavaScript initié en première année. Il sera présenté une architecture logicielle permettant la maintenance et l’évolution de la gestion d’interfaces Web complexes, spécialement dans le contexte d’une programmation événementielle.

L’enseignement est constitué de 7 séances de cours, 3 séances de travaux dirigés et 15 séances de TP. L’accent est donc mis sur la pratique sur ordinateur, seul ou en binôme.

Type Volume horaire
Cours magistraux (CMs) 7 x 1h30 (10h30)
Travaux dirigés (TDs) 3 x 1h30 (4h30)
Travaux pratiques (TPs) 15 x 1h30 (22h30)

À la fin de cette UE, vous serez capable de :
  1. Développer des interfaces Web complexes en HTML et CSS ;
  2. Maîtriser les fondements du langage JavaScript ;
  3. Gérer la modification de ces interfaces en fonction des interactions avec l’utilisateur ;
  4. Structurer l’architecture d’une application conséquente permettant sa maintenance et son évolution.
Poursuites possibles : Conception d’Applications Internet (CAI) en semestre 5 (L3).
