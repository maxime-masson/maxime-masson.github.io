Tout est objet en Javascript§

En Javascript, toutes les variables contiennent des objets, de nature différente.

Tout objet contient

  • un constructeur : qui initialise ses propriétés avec des paramètres
  • des propriétés : qui sont des variables contenues dans l’objet
  • des méthodes : qui sont des fonctions contenues dans l’objet et pouvant modifier ses propriétés

On distingue :

  • les objets natifs fournis par le langage Javascript
  • des objets supplémentaires créés par des développeurs en composant les objets natifs
3

Exemple avec l’objet string§

string est l’objet qui gère les chaînes de caractères.

let message = 'Hello World';   // construction d'une chaîne de caractère

console.log( message.length ); // affichage de la propriété length (longueur de la chaîne)

console.log( message.toUpperCase() ) // appel de la méthode qui renvoie la chaîne en majuscule

// Affiche :
// 11
// HELLO WORLD

Le point (comme dans message.length, ou message.toUpperCase()) est le symbole qui permet d’accéder aux propriétés et aux méthodes d’un objet.

Déclaration, initialisation d’un tableau et accès à ses éléments§

Un tableau est un objet particulier qui permet de stocker des valeurs associées à un indice.

Il se déclare et s’intérroge avec l’opérateur crochet [].

let fruits = ['pomme', 'poire', 'banane'];  // initialise un tableau avec 3 éléments

console.log( fruits.length );  // affiche le nombre d'éléments
console.log( fruits[0] );      // affiche premier élément du tableau

fruits[2] = 'orange';  // on remplace banane par orange
fruits[4] = 'raisin';  // on ajoute un élément à la cinquième position

console.log( fruits ); // affiche le tableau

Affiche :

['pomme', 'poire', 'orange', undefined, 'raisin']

Parcourir un tableau avec les méthodes forEach() et map()§

Les méthodes forEach() et map() des objets Array appliquent une fonction à chacun des éléments.

forEach() ne fait qu’appeler la fonction alors que map() retourne un nouveau tableau :

let fruits1 = ['pomme', 'poire', 'banane'];

fruits1.forEach( function(value, index, array) {           // fonction anonyme à 3 arguments
  console.log('l\'élément ' + value + ' se trouve à l\'indice ' + index);
});

let pronoms = ['ma', 'ta', 'sa'];
let fruits2 = fruits1.map( (v,i,a) => pronoms[i] + ' ' + v );            // fonction fléchée
console.log(fruits2);

"l'élément pomme se trouve à l'indice 0"

"l'élément banane se trouve à l'indice 2"

Filtrer les éléments d’un tableau avec la méthode filter()§

Seul les éléments pour lesquels la fonction renvoit true sont gardés :

let fruits1 = ['pomme', 'poire', 'banane', 'kaki'];

// ne gardons que les éléments de moins de 6 caractères
let fruits2 = fruits1.filter( function (value, index, array) {  // fonction anonyme
  return value.length < 6;
} );
console.log(fruits2);

let fruits3 = fruits1.filter( (v,i,a) => (v.length < 6) );      // fonction fléchée
console.log(fruits3);

// ne gardons que les éléments qui contiennent 'po'
let fruits4 = fruits1.filter( (v,i,a) => (v.includes('po')) );
console.log(fruits4);

['pomme', 'poire', 'kaki']
['pomme', 'poire', 'kaki']
['pomme', 'poire']

Réduire un tableau avec la méthode reduce()§

Réduire un tableau de nombres en un nombre

let notes = [15, 6, 12, 17, 8];

let somme = notes.reduce( (accumul, value, index, array) => {
  return accumul + value;
});
console.log(somme);    //  --> 58

let moyenne = notes.reduce( (acc,v,i,a) => (acc + v) ) / (notes.length || 1);

console.log(moyenne);  //  --> 11.6

Réduire un tableau de chaînes de caractères en une chaîne

let fruits1 = ['pomme', 'poire', 'banane', 'kaki'];

let fruits2 = fruits1.reduce( (acc,v) => (acc+'/'+v) );  // eq à fruits1.join('/')

console.log(fruits2);  //  --> 'pomme/poire/banane/kaki'

Décomposer et déstructurer un tableaux§

L’opérateur de décomposition ... transforme un tableau en une suite d’éléments :

let notes1 = [15, 6, 12, 17, 8];
let notes2 = [14, 16];
let notes3 = [...notes1, ...notes2];  //  --> [15, 6, 12, 17, 8, 14, 16]

let noteMax;
noteMax = Math.max(notes3);    // Erreur ! max ne prend pas un tableau en argument
noteMax = Math.max(...notes3); // Par contre il peut prendre plusieurs arguments
console.log('max : '+noteMax);

let [premier, deuxième, ...reste] = notes3;
console.log('1er = '+ premier +', 2ème = '+ deuxième +' et le reste = '+ reste);

[deuxième, premier] = [premier, deuxième]   // échanger des valeurs sans 3ème variable !
console.log('1er = '+ premier +', 2ème = '+ deuxième);

'max : 17'
'1er = 15, 2ème = 6 et le reste = 12,17,8,14,16'
'1er = 6, 2ème = 15'

Un tableaux comme valeur de retour d’une fonction§

En renvoyant un tableau, une fonction peut retourner plusieurs résultats :

let notes = [15, 6, 12, 17, 8];

function minmax1(arr) {
  let min = Math.min(...arr);
  let max = Math.max(...arr);
  return [min, max];
}

function minmax2(arr) {
  return [Math.min(...arr), Math.max(...arr)];
}

let minmax3 = (arr) => [Math.min(...arr), Math.max(...arr)];

console.log( minmax3(notes) );

let familleAddams = {

let fils  = familleAddams.enfants[0];
let fille = familleAddams.enfants[1];
console.log( 'le fils : '+ fils +' et la fille : '+ fille);

Définition de méthodes dans un objet littéral§

Des fonctions peuvent êtres associées aux clés d’un objet. On les nomme alors les méthodes de l’objet :

let coord = {
  x: 0,  // initialisé à 0 par défaut
  y: 0,

  init: function (x, y) {    // on associe une fonction à la clé init
    this.x = x;   // this désigne l'objet courant !
    this.y = y;
  },              // notez la virgule obligatoire entre les propriétés !

  position() {    // méthode avec une syntaxe plus courte eq. à position: function() {...}
    return [this.x, this.y];   // renvoie ici un tableau à deux éléments
  },
};

coord.init(2,3);
console.log( coord.position() );

[2, 3]

Attention à la copie superficielle des objets !§

L’affection d’un objet à un autre créé les mêmes clés mais recopie les références vers les valeurs :

let coord = {
  x: 0,
  y: 0,
  init(x, y) {
    this.x = x;
    this.y = y;
  },
  position() {
    return [this.x, this.y];
  } ,
};
coord.init(2,3);
console.log( coord.position() );   // --> [2, 3]

let coord2 = coord;  // attention copie superficielle ! Autre nom pour le même objet !
coord2.x = 4;
console.log( coord2.position() );  // --> [4, 3]
console.log( coord .position() );  // --> [4, 3]

Conséquence sur le passage d’objet comme arguments de fonctions§

À savoir : la modification d’un argument objet passé à une fonction modifie l’objet original :

let coord = {
   // ...
};
let number = 2;
let array  = [2,3];
coord.init(2,3);
console.log(number, array, coord.position() );

function change(num, arr, obj) {
  num    = 1;
  arr[0] = 1;    // Éviter de modifier un tableau passé en argument
  obj.x  = 1;    // Éviter de modifier un  objet  passé en argument
}
change(number, array, coord);
console.log(number, array, coord.position() );

2 [2, 3] [2, 3]  // valeurs initiales
2 [1, 3] [1, 3]  // valeurs modifiées par la fonctions pour le tableau et l'objet

utilisation des objets littéraux§

Les objets littéraux sont bien adaptés quand on a besoin d”un exemplaire d’une structure qui rassemble des propriétés et des méthodes.

Quand on manipule de nombreux exemplaires d’une telle structure, il faut passer par un mécanisme de création d’objets basés sur le même modèle. On parle alors d”instances d’une même classe.

Objets comme instances d’une classe§

26

Définition d’une classe et déclaration d’une variable de type class§

Des objets sont fabriqués (instanciés) à partir d’un modèle (une classe). Une instance est créée à la déclaration en appelant le constructeur avec ses arguments.

class Coord {
  constructor (x,y) {    // Un constructeur par classe et ses arguments
    this.x = x;          // ici, créé les propriétés x et y
    this.y = y;          // et les initialise avec les arguments
  }
  position() {                   // une méthode
    return [this.x, this.y];     // qui renvoie un tableau
  }
}

let c1 = new Coord(2,3);         // déclare deux instances de la classe Coord
let c2 = new Coord(4,5);         // qui sont construites avec les valeurs passées

console.log(c1.position(), c2.position());

[2, 3] [4, 5]

Exemple d’utilisation d’objets pour les interfaces Web§

Définition de la classe HtmlList :

class HtmlList {
  constructor (id, type, items) {
    this.type = type;                            // le type de la liste ('ul' ou 'ol')
    this.items = items;                          // les items de la liste sous forme de tableau
    this.element = document.getElementById(id);  // connecte la liste à un élément du document
  }
  render () {                                    // génération du HTML
    let html = '<'+this.type+'>\n';
    html += this.items.map( (v) => ('  <li>'+v+'</li>') ).join('\n');
    html += '\n</'+this.type+'>';
    this.element.innerHTML = html;                // injectin du HTML dans la balise connectée
  }
  push (item) {
    this.items.push(item);
  }
}

ul.push('abricot');  // ajoute un fruit à la liste d'items de l'objet
ul.render();         // modifie la liste ul affichée dans la page web !

Les classes Javascript§

C’est un ajout récent au langage pour cacher le méchanisme sous-jacent de Javascript pour la création d’objets sur la base de prototypes.

Elles permettent de mettre en place les notions classiques de la programmation orientée objet (héritage, polymorphisme) :

class UlList extends HtmlList {    // hérite des propriété et méthodes de la classe HtmlList
  constructor (id, items) {
    super(id, 'ul', items);        // appelle le constructeur de la classe héritée
  }
}

let ul2 = new UlList('fruits', ['orange']);    // nouvelle liste à puce attachée au div#fruits
ul2.push('citron');                            // utilise les méthodes héritées
ul2.render();                                  // la liste à puce contient maintenant :
                                               //   * orange
                                               //   * citron

Les littéraux de gabarit, chaînes de caractères multi-lignes§

C’est un autre moyen de construire des chaînes de caractères, entourées du caractère  » `  » (back quote).

  • ils sont multi-lignes
  • on peut injecter des chaînes de caractères issues de variables ou d’expressions dans ${var|exp} .
render () {                        // avec chaines de caractères
  let html = '<'+this.type+'>\n';
  html += this.items.map( (v) => ('  <li>'+v+'</li>') ).join('\n');
  html += '\n</'+this.type+'>';
  this.element.innerHTML = html;   // injectin du HTML dans la balise connectée
}

render () {                        // avec littéraux de gabarit
  let html = `<${this.type}>
  ${ this.items.map( (v) => (`<li>${v}</li>`) ).join('\n') }
  </${this.type}>`;
  this.element.innerHTML = html;   // injectin du HTML dans la balise connectée
}

Le format JSON§

JSON est un format pour stocker des données structurées dans une chaine de caractères.

Ce format est souvent utilisé pour transporter des données entre un serveur et une page Web.

JSON signifie JavaScript Object Notation 

// Exemple de données JSON
const data_string1 = '{"name": "Hochon", "age": 18, "address": ["2 rue du soir", "64000", "Pau"]}';

Une chaîne de caractères JSON peut être convertie en un objet Javascript (et vice versa) :

let person1 = JSON.parse( data_string1 );
console.log('nom de la personne : '+person1.name);  // affiche "Hochon"
person1.firstname = 'Paul';                         // ajout d'une propriété à person1

const data_string2 = JSON.stringify(person1);       // on convertit l'objet en une chaîne JSON
console.log(data_string2);
// affiche : {"name": "Hochon", "age": 18, "address": ["2 rue du soir", "64000", "Pau"], "firstname": "Paul"}

JSON : quelques contraintes§

À savoir :

  • Une chaîne JSON ne contient que des données, pas de méthodes (ou fonctions)
  • Utiliser des guillemets autour des propriétés et valeurs, pas des apostrophes
  • toutes les propriétés doivent être entre guillemets.

Trouvez les erreurs dans cette chaîne JSON :

{ "'name': "Hochon", age: 18, "address": ["2 rue du soir", "64000", "Pau"],
  firstname(){return "Paul"} }
