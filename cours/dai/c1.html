Canevas d’une page Web§

<!DOCTYPE HTML>
<html>
<head>                         <!--- Entête ------------------------------>
  <meta charset="utf-8" />
  <title>Abonnement</title>
  <link href="./style.css" rel="stylesheet" />
</head>

<body>                         <!--- Corps ------------------------------->
  <h1>Inscription</h1>
  <form method="get" action="https://httpbin.org/get">
    <label>Email : </label>
    <input type="text" />
    <br />
    <button type="submit" onclick="return valideForm();">S'abonner</button>
  </form>

  <script type="text/javascript" src="./validation.js"></script>
</body>
</htm>

Notions acquises en HTML§

  • Anatomie des balises

    <balise attribut1="val1" attribut2="val2">Contenu</balise>

  • Respect de l’imbrication des balises

    <p>Un texte avec du <strong>gras</strong> et de l'<em>italique</em>.</p>

  • Inclusion de ressources avec les bons chemins

    <link  href="./styles/basic.css"  rel="stylesheet" />
<script src="./js/lib.js"        type="text/javascript"></script>

Notions acquises en HTML§

  • Bonne structuration d’une page Web

    • liens :

      <a href="http://www.domaine.com/">Lien vers domaine</a>

    • Listes à puce :

      <ul><li>item1</li><li>item2</li></ul>

    • tableaux :

      <table><tr><td>l1c1</td><td>l1c2</td></tr>
<tr><td colspan="2">l2c1</td></tr></table>

    • etc.

Notions acquises en Javascript§

  • événements associés aux balises qui déclenchent des fonctions :

    <p onmouseover="réagit()">Paragraphe sensible à la souris.</p>

  • fonctions pour manipuler le BOM (Browser Object Model) :

    window.history.back();
console.log(window.location);
let onglet = window.open('http://www.youtube.com');
window.alert('Onglet ouvert !');
window.document.write("<b>On réécrit le document !</b>");

  • fonctions pour manipuler le DOM (Document Object Model) :

    let ulTag = document.getElementById('list');
let item  = document.createElement("li");
item.innerHTML = "Lorem Ipsum";
ulTag.appendChild(item);
