Vous implémenterez les défis proposés dans le cours sur les bases du langage Javascript.

Exercice 2 - Boucle while

Dans des fichiers exo2.html et dixchiffres_while.js, modifiez le code suivant : 

let message = '';
let i = 0;
while (i < 10) {
  message = message + i + ', ';
  i = i + 1;
}
alert(message);

Afin qu’il affiche les 10 chiffres par ordre décroissant de 9 à 0, et sans virgule finale !

Exercice 3 - Boucle do ... while

Dans des fichiers exo3.html et controle_saisie.js, transformez le code suivant : 

let nombre;
do {
  nombre = prompt('Entrez un nombre supérieur à 10');
} while (nombre <= 10);
alert(nombre + ' est bien supérieur à 10 !');

Afin de s'assurer de la saisie d’un nombre entier et impair compris dans l’intervalle [10; 20].

Indice : Utilisez la fonction parseInt et l’opérateur % (modulo).

Testez avec les valeurs : 12, 8, 2.5, toto et 15 par exemple.
